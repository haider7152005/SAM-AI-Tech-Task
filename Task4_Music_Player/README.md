# Task 4 - Music Player

## Description

A Music Player Web Application developed using HTML, CSS, and JavaScript. The application allows users to play, pause, skip to the next track, and return to the previous track while providing a simple and interactive user interface.

## Features

* Play Music
* Pause Music
* Next Song
* Previous Song
* Playlist Support

## Technologies Used

* HTML
* CSS
* JavaScript
