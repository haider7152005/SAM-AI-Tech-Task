# Task 2 - To-Do List Application

## Description

A To-Do List Web Application built using HTML, CSS, and JavaScript. The application allows users to add, delete, and mark tasks as completed. Local Storage is used to save tasks permanently in the browser.

## Features

* Add Tasks
* Delete Tasks
* Mark Tasks as Completed
* Local Storage Support

## Technologies Used

* HTML
* CSS
* JavaScript
